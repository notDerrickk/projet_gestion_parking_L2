
# Projet Java Parking

## Description

Projet Java de gestion de parking : administration des places, abonnés et tarifs à l’aide d’une structure orientée objet.
L’application permet de modéliser un parking, ses places disponibles, ses abonnés et la gestion des tarifs d’abonnement.

## Fonctionnalités principales

* Gestion des **places de parking** (disponibles, occupées, etc.)
* Gestion des **abonnés** et de leurs informations
* Calcul et suivi des **tarifs d’abonnement**
* **Administration** du parking (ajout, suppression, mise à jour des données)


## Tests

Des tests unitaires sont réalisés avec **JUnit** pour garantir le bon fonctionnement des classes principales.
